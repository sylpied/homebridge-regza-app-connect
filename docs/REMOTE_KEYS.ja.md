# REGZA リモコンキー参考表

[English](REMOTE_KEYS.md) | 日本語

この一覧は提供されたREGZAリモコンコード表を転記した参考資料です。機種・ファームウェアによって動作が異なる可能性があります。**「確認済み」以外のキーは本プラグインで実機確認していません。** 録画やディスクなど状態を変更するキーは特に注意して試してください。

## レコーダー

### DBR-M590

DBR-M590はテレビ用の6桁コードではなく、レコーダー用の短縮コードを`/remote/remote.htm?key=`へ渡します。

| 操作 | コード |
|---|---|
| 電源トグル | `12` |
| スタートメニュー | `46` |
| 決定 / 戻る / 終了 | `44` / `4b` / `60` |
| 上 / 下 / 左 / 右 | `c0` / `c8` / `cc` / `c4` |
| 再生 / 一時停止 / 停止 | `13` / `17` / `16` |
| 早戻し / 早送り | `9a` / `98` |
| 前 / 次スキップ | `84` / `80` |
| 地デジ / BS / CS | `bd` / `be` / `bf` |
| 録画 / 録画リスト | `15` / `6d` |
| 画面表示 | `5a` |
| タイムスリップ | `1a` |

電源、スタートメニュー、下、決定はDBR-M590実機で確認済みです。`1a`はAPK上では`TIME_SLIP`であり、過去番組表を開くキーではありません。DBR-M590の最初の決定操作には、確認済みのスタートメニュー`46`を使用します。
